# Windows OS Architecture, NTFS, and User Permissions Lab

## Objective
Understand the foundational components of the Windows operating system, focusing on file systems, user accounts, permissions, and core OS directories from a security perspective.

## Lab Overview
This lab explores Windows editions, the NTFS file system, user account types, permissions, and critical system folders such as System32. The lab emphasizes how these components impact system security and access control.

## Key Topics Covered
- Windows editions (Home vs Pro vs Server)
- NTFS features (permissions, encryption, journaling)
- User accounts, profiles, and group permissions
- System directories (C:\\Windows, System32)
- User Account Control (UAC) fundamentals

## Skills Demonstrated
- Windows OS navigation and configuration
- NTFS permission analysis
- User and group management
- Security implications of system-level access

## Security Relevance
These concepts are essential for SOC analysts and cybersecurity professionals when investigating privilege misuse, unauthorized access, and system compromise.
