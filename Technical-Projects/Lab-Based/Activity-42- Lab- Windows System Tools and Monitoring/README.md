# Windows Administrative Tools, Monitoring, and Troubleshooting Lab

## Objective
Explore built-in Windows administrative and monitoring tools used for system diagnostics, performance analysis, and troubleshooting.

## Lab Overview
This lab focuses on Windows utilities such as System Configuration, Computer Management, Event Viewer, Task Scheduler, Resource Monitor, and Command Prompt. Emphasis is placed on how these tools support system visibility and operational monitoring.

## Key Topics Covered
- MSConfig and startup troubleshooting
- Event Viewer and Windows logs
- Task Scheduler and automated tasks
- Resource Monitor (CPU, memory, disk, network)
- Administrative command-line tools (ipconfig, netstat, net, whoami)

## Skills Demonstrated
- System diagnostics and performance monitoring
- Log inspection and operational awareness
- Command-line usage for system and network analysis

## Security Relevance
These tools are frequently used in SOC environments for alert investigation, incident response, and detecting abnormal system behavior.
