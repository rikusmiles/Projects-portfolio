# Windows Security Features, Firewall, and Endpoint Protection Lab

## Objective
Understand built-in Windows security controls and how they protect systems against malware, unauthorized access, and data loss.

## Lab Overview
This lab examines Windows Security features including Windows Defender, Firewall profiles, BitLocker, Windows Updates, User Account Control, and Volume Shadow Copy Service (VSS). The focus is on endpoint protection and system hardening.

## Key Topics Covered
- Windows Defender and real-time protection
- Firewall profiles (Domain, Private, Public)
- Windows Updates and patch management
- BitLocker drive encryption concepts
- Volume Shadow Copy Service (VSS) and ransomware recovery
- Living-off-the-Land (LOLBins) awareness

## Skills Demonstrated
- Endpoint security configuration
- Firewall and network protection understanding
- Security hardening and risk mitigation

## Security Relevance
These controls form the foundation of endpoint security and are critical for defending against malware, ransomware, and unauthorized network access.
